# D5 Render Documentation

Official documentation for [D5 Render](https://www.d5render.com/).

## Overview

This repository contains the source files for the D5 Render knowledge base, powered by **[Mintlify](https://mintlify.com)**.

## Directory Structure

```
docs/                    # Documentation content (Markdown / MDX)
├── getting-started/     # Quick start & workflow
├── user-guide/          # Full user manual
├── workflow/            # Plugin integrations (SketchUp, Revit, Rhino, etc.)
├── d5-account-and-purchase/
├── d5-lite/             # D5 Lite documentation
├── instruction-for-d5-for-teams/
└── learn-more/
mint.json                # Mintlify site configuration
```

## How It Works

Content updates are pushed to this repository via automated workflows (Feishu Agent → format & translate → push). Mintlify automatically builds and deploys the documentation site on each update.

## Contributing

This is a managed documentation repository. For content changes, please contact the documentation team.
