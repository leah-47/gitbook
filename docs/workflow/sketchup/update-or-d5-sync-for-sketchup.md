# Update | D5 Sync for SketchUp

## Ver. 0.8.4.0003

Release date: 20230504

Now we provide two versions which have different solutions on light sync.\
For details, please read this topic: [Light group changes and different versions for downloading | D5 Converter for 3ds Max and SketchUp 44](https://forum.d5render.com/t/light-group-changes-and-different-versions-for-downloading-d5-converter-for-3ds-max-and-sketchup/15616)

### **Fixes**

1. For solid color materials which do not have textures, the UV will not be correctly sync-ed.
2. Error opening recovered files in SketchUp.
3. A large file with many components imported into D5 engine will display abnormal highlights and purple light.
