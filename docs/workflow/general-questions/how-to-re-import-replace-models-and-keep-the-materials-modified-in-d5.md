# How to re-import/replace models and keep the materials modified in D5?

<div align="left"><figure><img src="/files/q32esthDEAQy8XyeUgBo" alt=""><figcaption></figcaption></figure></div>

## Replace from local

Click the "Replace from local" button and select a model file from the local folder to replace the model in the current scene.

## Reload&#x20;

After the model source file has been modified and saved, click reload and D5 will automatically sync the model to be up-to-date and consistent with the source file.
