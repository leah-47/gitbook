# Archicad

#### Download and Installation <a href="#tt2ql" id="tt2ql"></a>

You can download the latest version on the official website, or download from the Welcome Page > Workflow.

After downloading, double-click the .exe file and follow the instruction to install D5 Converter-ARCHICAD.

Supported versions: **Archicad 21 to 28, Archicad Solo and Archica**d SE

#### One-click to launch D5 Render <a href="#atmbl" id="atmbl"></a>

<img src="/files/-MkYRSrPIkTgciLY51cP" alt="" data-size="line"> Start D5 Render with a single click of the button and establish a connection with the current model in Revit.

<img src="/files/-MkYRwsw3sj31PMpBolk" alt="" data-size="line"> Turn off synchronization and disconnect D5 Render from the current model.

![](/files/-MkbXDzQAlm5ooBPMWbZ)

**Create a project:** create an empty. drs scene and import the .rvt file from the current Revit to establish a connection with D5 Render

**Associate a project:** associate the current .rvt file in Revit with the model file in the existing .drs scene

![](/files/-MkbW0FD9La6JyEjt2h1)

When a model file already exists in D5 Render, the first time you synchronize it, you need to connect the synchronized model to an existing model in the resource list or create a new model.

By default, D5 Render is closed. First, you need to click<img src="/files/-MkYRSrPIkTgciLY51cP" alt="" data-size="line">to start D5 Render and choose to create a project or associate a project, to active D5 Converter.

If D5 Render is already started, you need to choose to create a new model or select the imported model linkage in the pop-up window.

#### Sync <a href="#yoqvc" id="yoqvc"></a>

<img src="/files/-MkYT1emzDsDMVcxxVaV" alt="" data-size="line"> Synchronize function, synchronize current model and corresponding material by incremental updates, which makes it faster to read Model Files.

It is suitable for changing the material and model by using the synchronization function when adjusting the details of the scheme.

#### View synchronization <a href="#d8ypd" id="d8ypd"></a>

<img src="/files/-MkYTsdr8HTi0icyuKQ7" alt="" data-size="line"> Turn on View Sync, the current view of ARCHICAD is synchronized with the view of D5 Render

<img src="/files/-MkYV9uzrzwHIK9X90DO" alt="" data-size="line"> Turn off View Sync, disconnect view sync from ARCHICAD and D5 Render

#### Scene list synchronization <a href="#vvbhr" id="vvbhr"></a>

<img src="/files/-MkYVQUG_mUxWmcIjnk2" alt="" data-size="line"> Synchronize the scene list in ARCHICAD to D5 Render with one click
