# Workflow | D5 Sync for Rhino

.3dm files can be imported into D5 Render directly, while D5 Sync for Rhino helps you to achieve real-time synchronization between Rhino and D5.



## Download & Installation <a href="#toc-h2-download-installation" id="toc-h2-download-installation"></a>



#### Supported versions

Rhino 6.1 and above

#### Download

Double click on the .rhi file after the download, and follow the instructions to install [D5 Sync for Rhino](https://github.com/d5render/D5_Converter/releases/download/latest/D5_Converter_Rhino_usa_latest.zip)

## Features <a href="#toc-h2-features" id="toc-h2-features"></a>

#### One-click to launch D5 Render in Rhino <a href="#toc-h3-one-click-to-launch-d5-render-in-rhino" id="toc-h3-one-click-to-launch-d5-render-in-rhino"></a>

* Import the new models into D5 Render to start real-time rendering&#x20;
* When a model file needs to be modified during rendering, start sync to create a connection with the original model or replace it with the new model

## Model/Material/Scene Sync <a href="#toc-h3-modelmaterialscene-sync" id="toc-h3-modelmaterialscene-sync"></a>

* After the current model has been edited in Rhino, quickly sync the new model and materials to D5 Render by the Sync button, while keeping the materials and scene parameters that had been adjusted in D5 Render.&#x20;
* Support the mapping of PBR materials for Rhino 7
* Support the mapping of basic materials for Rhino 6
* Support the sync of VisualARQ data
* Support the sync of all scenes from NamedView

## View Sync <a href="#toc-h3-view-sync" id="toc-h3-view-sync"></a>

* Keep the views of Rhino and D5 Render consistent when syncing
* Support one-click to turn view sync on or off

## Light Sync <a href="#toc-h3-light-sync" id="toc-h3-light-sync"></a>

* Support the sync of Spotlight, Point light, Rectangular light, and Linear light.
