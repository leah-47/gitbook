# Common issues | D5 Sync for Rhino

## D5 Converter-Rhino shows '' unknown error''

#### There are the following reasons and solutions:

1. Rhino is unable to connect to the network. It is suggested to review the firewall settings in your Windows system and either disable the firewall or remove the block on Rhino's network access.
2. It is advisable to change the installation directory of D5 Render from a folder that has Chinese characters to a folder entirely in English.

## D5 Converter-Rhino has been successfully installed, but it does not show up in Rhino?

#### There are the following reasons and solutions:

1. Confirm that the "D5Converter" is enabled in ''Tools''-''Options...''Plug-ins'';
2. Enable the ''D5 Converter'' toolbar in "Tools" - "Toolbar Layout".
