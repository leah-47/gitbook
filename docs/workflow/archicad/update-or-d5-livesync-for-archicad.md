# Update | D5 LiveSync for Archicad

## Ver. 1.2.0.0002

`Release date: 20251022`

#### **New**

* Supports Archicad 29
* Supports Archicad parallel projection
* Supports more Archicad light mappings, including object-light combinations

#### **Optimization**

* Improved large scene merging to fix black or distorted screens when syncing to D5
* Improved selection export; windows and walls can now be selected separately

#### **Fixes**

* Fixed curtain wall instance errors caused by model matrix issues
* Fixed occasional Archicad crashes during live sync

## Ver. 1.1.0.0010

`Release date: 20250422`

#### **New**

* Supports group export (preserved layer structure)
* Supports export selected objects

**Optimization:**

* Optimized linkage performance

**Fixes:**

* Some anomalies caused by different system regions
* Some large scenes are exported very slowly

## Ver. 1.0.0.0006

`Release date: 20241230`

#### **New and Optimization**

* Support Archicad 28
* Support Archicad-D5 real-time sync
* Added Sync once option
* Support Spanish, French, and Japanese
* Support Thumbnail Display for .d5a Export
* Recorded user preferences for single send or real-time sync, defaulting to the last selection for the next linkage
* Optimized UI
