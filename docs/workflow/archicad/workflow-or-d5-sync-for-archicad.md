# Workflow | D5 Sync for Archicad

## Supported versions

Archicad21 - Archicad 27

## Download & Installation

Double click on the .rhi file after the download, and follow the instructions to install [D5 Sync for ArchiCAD](https://www.d5render.com/workflow/archicad?_sasdk=fMjIyNTc)

## Features 

<div align="left"></div>

#### One-click to launch D5 Render in Archicad 

* Import the new models into D5 Render to start real-time rendering
* When a model file needs to be modified during rendering, start sync to create a connection with the original model or replace it with the new model
* Import ArchiCAD models into D5 Render by Sync feature without exporting d5a

#### Model/Material/Scene Sync

* After the current model has been edited in ArchiCAD, quickly sync the new model and materials to D5 Render by the Sync button, while keeping the materials and scene parameters that had been adjusted in D5 Render
* Sync the scene list in ArchiCAD to D5 Render

#### View Sync

* Keep the view of ArchiCAD and D5 Render consistent in the horizontal direction
* Support one click to turn view sync on or off

#### Light Sync

* Click on the Sync Light button to synchronize Archicad's model light sources with D5 render.
* Click the Sync Light button to sync Archicad's pure light sources to the D5 render, including light source types (point light, spotlight, strip light, rectangular light), positions, sizes (for rectangular/strip light).

#### **Export d5a files**

* ArchiCAD files can be imported D5 Render by converting to .d5a files through D5 Converter

Settings

* Textures Folder: Customize the storage location for all texture resources used in the current Archicad file.
