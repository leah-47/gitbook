# The imported model in D5 is a whole. How to separately adjust individual objects?

Currently the model is imported into D5 as a whole and relies on the materials to differentiate.&#x20;

**It is recommended to** ********differentiate the materials in the modelling software before importing the model into D5******. A**fter importing into D5, you can use the shortcut key "i" to quickly pick up the material for editing, or you can lock the model in the left sidebar--object list, and click on the model for material editing.
