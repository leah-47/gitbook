# How to use Support Tool?

D5 Support Tool, which can check the network conditions, hardware, software conflicts, and also collect logs and make it easier for troubleshooting.

#### Ways to Enter D5 Support Tool and Collect Log Files

1. Click "Help"-"Collect Log Files" in **D5 Launcher** and then click "**Collect Logs**"

<figure><img src="/files/2qn5H8tbROeeSKnuQIBB" alt=""><figcaption><p>Support Tool</p></figcaption></figure>

2. Click “Help” in the menu bar, then “System info” and “**Collect Log Files**”.

<figure><img src="/files/xXcpYjVwyuh2QNN1oEmT" alt="" width="563"><figcaption><p>Support Tool</p></figcaption></figure>

3. Open the D5 Render installation path, Double click the “**Support Tool**”.

<figure><img src="/files/JkzinQPudLnwvWM37ROK" alt="" width="563"><figcaption><p>Support Tool</p></figcaption></figure>

After entering D5 Support Tool, “**Check All and Zip At Once**”, then send the exported log file (which is called **zziplog** in D5 folder) to D5 Support team.
