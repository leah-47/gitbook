# Why are close textures clearer but distant textures blurry in modeling software/D5?

Specific issues: Modeling software/D5 shows clearer textures in the nearer areas and blurred textures in the farther areas.

<div align="left"><figure><img src="/files/zi0jbkUoma76989X9F20" alt=""><figcaption><p>Effect of incorrect settings in modeling software/D5</p></figcaption></figure></div>

Reason: Select the "NVIDIA Control Panel-3D Settings-Use the advanced 3D image settings", which may make modeling software/D5 show blurred textures in the farther areas.

Solution: The correct setting is select the "Let the 3D application decide" instead of  "NVIDIA Control Panel-3D Settings-Use the advanced 3D image settings".
