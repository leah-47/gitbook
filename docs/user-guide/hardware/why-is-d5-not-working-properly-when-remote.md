# Why is D5 not working properly when remote?

Specific problem performance:

* The right mouse button does not move the viewport left or right.
* Lights can only be placed in the upper left corner.
* Objects in the scene cannot be selected.

Suggested solution:&#x20;

1. When remote, it is recommended that the mouse, keyboard and monitor are linked and turned on.
2. Turn off the Hardware Acceleration GPU program in Windows System Settings - Screen - Graphics Settings.
3. Problems with the monitor screen: e.g. Dell monitors do not work properly when turned off.
4. Interface cable problems: the use of DP interfaces when the display is closed 3D software can not be operated or can not see the interface; while the general use of HDMI interfaces can be displayed normally
