# How to use full screen mode?

<mark style="background-color:purple;">Use the</mark> <mark style="background-color:purple;"></mark><mark style="background-color:purple;">**shortcut key F9**</mark> <mark style="background-color:purple;"></mark><mark style="background-color:purple;">to hide all panels in one click and display the live window in full screen, while the window supports scaling to a smaller size.</mark>

> Note: Use **the shortcut keys F10\&F11** to hide the left and right sidebars respectively.

* In modelling, you can work with the Sync plugin to display modelling and rendering on a single screen at the same time, to enhance work efficiency.&#x20;
* In video editing, you can also display full screen with one click. Press the space bar to preview the video effect directly.
