# How to set grass? How to grow grass with one click?

Use the Grass material template to quickly create large areas of natural grass.

![](/files/-MkeuquObuEXjS4h44Jh)

The grass material parameters are as follows:

* **Base Color/Base Color Map:** Controls the color of the ground as well as the grass.

![](/files/-MkevBkY7UplZRYo7tcx)

> **Success:** It is recommended to use a grass texture map linked to the Base Color Map slot, the generated grass will automatically load the color of the base color map to generate a natural grassland.


* **Normal:** Controls the bumpy effect of the ground below the grass.&#x20;
* **Specular:** Controls the Reflectance of the grass and the ground.&#x20;
* **Roughness:** Controls the Roughness of the grass and the ground, which affects the reflection blur.&#x20;
* **Height:** Controls the height of the automatically generated grass.

![](/files/-MkevS2fO1p23Qlc6Wfl)

## FAQS

1. Why does the grass material become sparse along the model boundaries?

* The grass material algorithms of versions before 2.7 make it hard to align with the edges of small-area models. Users have to manually fill in the flawed areas with models from D5 Asset Library > Nature > Other Herbs, which can also be found by searching 'Grass Material'.
* D5 2.7 optimizes grass with better scattering and edge-cutting algorithms to precisely align with model edges. It also optimizes existing grass materials with brighter colors and a denser appearance in aerial view.

2. Why does the grass material become lifted in D5 2.7?

Models having chamfers or uneven surfaces may cause the grass to be lifted. In this case, please modify the model.

Please adjust this part of the model.

3. Why is the distant grass generated by the terrain grass material template not shown?

For real-time efficiency, the grasses in the farther distance will become pure colour in the preview, and will be shown as the actual terrain grass effect in the output, which will not affect the rendering results.&#x20;

In ‘Preference’ -> ‘Rendering’ -> ‘Cull Distance’, you can set the fading distance from the camera for the smaller online assets library models in the scene to achieve the desired effect.

<figure><img src="/files/pqOFIlN8YvRjYMriIYD2" alt=""><figcaption></figcaption></figure>

Preview Cull Distance: Support to set the fading distance within the preview, the range is 20-9999 metres. When the distance is large, it will cause real-time preview lag.&#x20;

Output Cull Distance: Support to set the culling distance when rendering images, the range is 20-9999 metres. When the distance is large, it will prolong the time of rendering out. To ensure the consistency between rendering and preview, the output cull distance must be greater than or equal to the real-time cull distance.&#x20;

> You can set different distances for real-time preview and rendering results separately, and this option follows the archive record.

4. Why the model can't scatter the grass?&#x20;

Please check the model faces before using the terrain grass material, terrain grass will only be generated on the positive side of the model normal.

5. Why is the terrain grass effect unusual in orthogonal views (top view/left view, etc.)?

This is a known issue with the current version: terrain grass is not displayed in orthogonal view (only the base color map is displayed).&#x20;

Please avoid using terrain grass in orthogonal views.

<figure><img src="/files/trIrhouufU10uTZ3sChb" alt=""><figcaption><p>Perspective View Effect &#x26; Orthogonal View Effect</p></figcaption></figure>

6. Why does it prompt ‘Fail to Apply, please confirm that each applied area is no larger than 25k㎡’?

The current version restricts the terrain grass area to a maximum of 25 square kilometers.
