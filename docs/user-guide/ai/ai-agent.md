# AI Agent



AI Agent portal has been added to the top menu bar, please click to enable it.

### 1. SmartPlanting

Enter your site location and preferences, and the AI Agent will utilize its plant database and climate model to instantly generate **a customized flower bed design plan tailored specifically to your site.** This approach ensures that the garden grows naturally based on the data stream. The design is optimized for areas up to 30m² (323 sq ft)



**Three Design Style Modes Now Available:**

* **Seasonal Flower Bed:** Designs optimized for specific seasonal viewing needs.
* **Natural Perennial Flower Bed:** Using perennials for a sustainable ecosystem, with plants of robust structures ensuring year-round beauty.
* **Minimalist Floralless Garden:** Uses diverse leaf shapes and subtle grayscale palettes to create tranquil beauty.

### 2. Design Assistant Toolkit



#### Export

Automatically export the plant list of all D5 online material libraries used in the scene, including key parameters such as name, size, classification, quantity, etc.

For the design scheme generated by AI flower border, a plant list with detailed information, such as habit, color, and season, can be additionally exported.



Supports uploading plant quotation files, with automatic cost matching and intelligent cost calculation.

* Only one file can be uploaded at a time (maximum size: 10MB).
* Supported formats: .xls and .xlsx.

#### AI Scene Match

Users only need to import the model and describe the picture's expectations, and AI will intelligently assist in the construction of light, shadow, and atmosphere.

**Use Method:**

* Describe the requirements to the AI design assistant, such as optimizing the atmosphere of the scene, adjusting the scene to the autumn dusk, enhancing the picture quality, adjusting the cold and warm colors, or changing the art style.
* AI generates a reference map from the current scene map and camera parameters, selects the most appropriate atmosphere for the scene, and automatically completes the optimization adjustment.
* The generated reference map can assist in scene construction. The materials in the reference map match similar models/materials in the material library, providing a key to improve the efficiency of scene construction.
* The reference graph supports semantic iterative generation, and it can be optimized according to further instructions.

#### AI Asset Recommendation

Supports image recognition and text retrieval.

**Use Method:**

* Upload the reference image or the reference image generated in the scene promotion agent to obtain the model matching recommendation and quickly locate the model material suitable for the scene in the material Library. The image size does not exceed 10MB.
* Enter keywords to quickly locate material and apply it to the scene.

Example: Enter "Please recommend the tree species suitable for the bird's eye view of the South", AI will provide the tree species suitable for the scene and display the corresponding model in the material Library, which is convenient and fast to apply to the scene and improve the creation efficiency.

#### Light Source Assisted Adjustment

Through deep learning and visual analysis technology, AI intelligent dimming can accurately understand the user's natural language instructions, and intelligently identify, locate, and adjust the parameters of the light source in the scene. This function is designed to address the problems of cumbersome lighting management and inaccurate parameter settings in complex scenes, and to support precise control of lighting in specific areas. It can also provide overall atmosphere optimization suggestions to prevent overexposure.

* Support for intelligent parameter adjustment of artificial light sources in the scene.
* The light source parameters can be quickly adjusted by semantic instructions to improve operation efficiency.



**Use Method:**



* Automatic adjustment: tell the AI design assistant through prompt words to optimize the specific light source type in the current viewport, such as a point light source, surface light source, disc light, etc., or a specific adjustment area. The Design Assistant triggers an optimization strategy to adjust the scene's lights as needed.
* Manual selection and adjustment: if a specific light source needs adjustment, you can accurately select the object to adjust with the selection tool, enter the prompt word, and the assistant will adjust the light in the scene according to the prompt word.
* *Example tuning command:*
  * *The right area of the scene is too dark, turn it up a little bit.*
  * *Please help me optimize the lights in the scene*
  * *Turn off all light sources in the living room*
  * *Adjust to colorful lighting atmosphere*

### 3. D5 Bot



D5 Bot is an AI-powered support that can answer questions about using D5 Render and provide smart solutions. For the most accurate help, please describe the issue or request in detail using text.

#### Issue Report in D5 Bot

* Added **Issue Report**: Supports automatic AI-generated summaries of issues and descriptions, along with manual screenshot uploads, to create reports ready for submission to the D5 Support team.
* After submission, the system automatically creates a support ticket and sends a response to the associated D5 account email, ensuring timely follow-up.
