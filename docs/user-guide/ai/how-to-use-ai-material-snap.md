# How to use AI Material Snap?

<figure><img src="/files/SpwYTUelPB39KA0mPUu1" alt=""><figcaption><p>AI Material Snap</p></figcaption></figure>

The ''AI Material Snap'' entry has been added to **Asset Library - Online - Material.** This feature supports uploading local images for material reference sampling. After uploading the reference image, users can select a specific area or use the entire image to generate materials. AI will intelligently analyze the selected image content and generate high-quality, directly applicable PBR materials with a single click. The generated materials can be applied directly to the current scene or saved to the local library for future use.

<figure><img src="/files/OAOMsSwO2E4QuACXlzDZ" alt="" width="563"><figcaption></figcaption></figure>

<figure><img src="/files/npsN2ZyQY1YB2h7oPKap" alt="" width="391"><figcaption><p>Add to Local/ Apply</p></figcaption></figure>

At the same time, D5 will also intelligently recommend similar-style materials from the online Asset Library based on the characteristics of the sampled image, thereby improving the efficiency of material creation and search.

***\*Note: The maximum supported image resolution for uploads is 6K.***
