# Why does custom lut not work?

You can check if the lut is a valid lut file in other software at first.&#x20;

If D5 is installed in a non-English path, the custom lut file may fail to read.

> Although the installation path is checked during installation, if you manually change the installation path to non-english after a successful installation, it will also cause this error.
