# How to save and reuse environment and post-effect presets?

1. In the right-click menu of the scene list, select "Create Preset", and fill in the preset name, **you can store the environment and effect parameters of the current scene as a preset in Studio**.&#x20;
2. The same preset parameters can be reused in other projects with one click, just open the Studio, click on the preset, and then the effects can be applied to the new scene.

> Note:
>
> * Environment and effect presets take up D5 Studio Cloud Space:  [What is Studio Cloud Space?](../studio/what-is-studio-cloud-space.md)
> * D5 Studio Cloud Space related rights can refer to: [Account Access to AI features](../ai/account-access-to-ai-features.md)



## FAQs

1. Why cannot create environment and effect presets?

> When users use D5 client version **CN_2.8.0.0710\&EN_2.8.0.0469**, if the HDRI in the scene is in exr format and save the environment parameter to Studio preset, it will prompt "File reading error, please check the file name".&#x20;

This problem has been fixed in **CN_2.8.0.0716 & EN_2.8.0.0471 (official version)**. It is recommended that users reinstall in a different path to prevent incomplete overwrite installation.

> **Info:** The fixed version is not actively pushed to the users with abnormal version, please use the latest version of the official website as the main.
