# How to use Outline mode in Style? What effects can be achieved?

<mark style="background-color:purple;">Turn on the outline mode can automatically depict the edge effect of the objects in the scene, you can set the colour of its line stroke, transparency, change the line width by distance and other parameters.</mark>&#x20;

* **Line**: Adjust the stroke colour and line transparency.&#x20;
* **Line Width**: Adjust the thickness of the stroke.&#x20;
* **Change width by distance**: The line width can be attenuated with the distance.&#x20;
* **Background Colour**: Adjusts the background colour of the overall screen of the outline mode.
