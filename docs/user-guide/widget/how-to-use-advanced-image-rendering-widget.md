# How to use Advanced Image Rendering Widget?

D5 Widget is a customized extension module of D5 Render, aiming to provide advanced features while keeping the simplicity.&#x20;

Go to Menu > Preference > Widget, and you can check what you installed, switch on/off with one click to manage your widgets.

<figure><img src="/files/dwzY1OIBsBEepITDXNKE" alt=""><figcaption><p>Advanced Image Rendering Widget</p></figcaption></figure>

After switching this widget, you can select .png .jpg .tga .tif .exr in the "Format" option, when rendering images.
