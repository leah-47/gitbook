# Vegetation Tool

In the Asset Library, numerous vegetation models are provided according to the classification of common plants, which can be downloaded by clicking on them, and can be used as a single model in the scene, or can be scattered in the scene using the vegetation tool.

## :paintbrush:Brush and Scatter

[Brush and Scatter](./vegetation-tool/brush-and-scatter.md)

:point_up_2:Use the Brush tool to draw the plants on the model surface.

## :broom:Eraser and Clean

[Eraser and Clean](./vegetation-tool/eraser-and-clean.md)

:point_up_2:Use the eraser to clear the drawn plants.

## :evergreen_tree:Vegetation Path

[Vegetation Path](./vegetation-tool/vegetation-path.md)

:point_up_2:Plant vegetation linearly by drawing paths.
