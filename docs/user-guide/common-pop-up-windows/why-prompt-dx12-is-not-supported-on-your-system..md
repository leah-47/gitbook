# Why prompt 'DX12 is not supported on your system. '?

<figure><img src="/files/oiISG2rmfFxaM2itqPj8" alt=""><figcaption><p>DX12 Error</p></figcaption></figure>

You may have installed an unsuitable driver for your graphics card. Go to NVIDIA's official website to re-download the graphics card driver and install it.
