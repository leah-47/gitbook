# Why prompt 'The material of model cannot be reused'?

<figure><img src="/files/y6BuTYHsdpOimvw7UldJ" alt=""><figcaption></figcaption></figure>

This popup is usually more likely to appear in the following cases:

1. The model position is far away from the origin position.
2. Select another material to assign to the vegetation.
3. The material is reused before it is loaded.
4. The model is not selected when applying a material after absorbing the material.
5. The model is a library model and does not support user-assigned materials.
6. Apply material to (shown/hidden) paths.
7. Materialize a section tool.
