# Why prompt 'Lights cannot exceed 4096'?



Currently, the maximum number of artificial and spontaneous lights in a viewport is 4096.&#x20;

If this limit is exceeded, it is recommended to manage the lights in each scene shot using layers or groups, and to hide or remove unnecessary lights for the rendered shots.
