# Why prompt 'The application could not start normally (0xc000007b)'?

<figure><img src="/files/ySPwxzt5Mu06IKH54XIv" alt=""><figcaption><p>0xc000007b Error</p></figcaption></figure>

It may be caused by the lack of D5 dependent system files in your computer. You can use the DXR Repair Tool to fix it.



If you still get errors after repairing, run D5 as Admin.
