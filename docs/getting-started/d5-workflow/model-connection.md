D5 Render provides two ways to connect with models: livesync plugins and direct file import options.

### Direct file import

Click the “Import” button on the top toolbar to import models.

D5 Render can directly read .skp / .fbx / .3dm / .abc / .d5a files.

<div align="left"></div>

### LiveSync with DCC software

Livesync plugins instantly mirror model changes into D5, making them a top pick for designers who often update their work and want to see their designs come to life in a very realistic setting.

D5 Render supports live sync with 3ds Max, Archicad, Blender, Cinema 4D, Rhino, Revit, SketchUp, and VectorWorks.
