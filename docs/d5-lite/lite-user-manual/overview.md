# Overview

### About D5 Lite

D5 Lite is a design tool made for space designers to use in the early stages of a project. It works smoothly inside SketchUp and other popular modeling software, helping designers speed up their workflow and show ideas faster.

D5 Lite combines AI and a rendering engine:

* The AI image-to-image feature runs in the cloud, and requires no local computing power. It can quickly and reliably generate images. It includes many built-in style presets for architecture, landscape, interior design, and more.&#x20;
* The rendering engine is built for a fast and simple workflow. It works as a plugin inside the modeling software and syncs lighting and material data in real time.

  D5 Lite is great for lightweight visual output in early design. All data can be sent to D5 Render for further detailed work.

### Instructions

1. Currently supports SketchUp 2021.1 to 2026.
2. D5 Lite requires an internet connection:

&#x20;      a. You need to log in to your D5 account. The account and login status are the same as in D5 Render.\
&#x20;      b. AI features rely on cloud computing, so a stable internet connection is necessary. If your network is poor, some features may not function properly. During peak times, you may experience delays.\
&#x20;      c. The asset library is online, so you need to download assets before using them.

3. D5 Lite is a **Beta** version. Features and performance will be improved based on user feedback.
4. Hardware requirements: Only NV RTX series graphics cards are supported. Intel and AMD graphics cards are not currently supported.
