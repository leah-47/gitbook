# Features



### Mode Switching

1. You can switch between the AI and Render modes using the button in the top-left corner.
2. The software will remember your last used mode and enter it by default the next time you start.

| AI Generation                                                                                       | Real-Time Rendering                                                                                 |
| --------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------- |
| <div></div> | <div></div> |
| Current mode: AI Generation                                                                         | Current mode: Real-Time Rendering                                                                   |
| Click the button to switch to Real-Time Rendering mode                                              | Click the button to switch to AI Generation mode                                                    |
