# How to use the D5 Showreel platform?

All Panorama Tours, Spatial Tours, and XR Tours uploaded by individuals or their teams are stored in the D5 Showreel, where you can view, manage, edit, or share tour files with others.

### Create cloud projects

**Currently, cloud projects can be created on two platforms: D5 Render client and D5 Showreel platform.**

**D5 Showreel**

* Select Team Projects(or Group Projects) → Click "Create Project"



**D5 Render client**

* Click comment → unbound cloud project → create cloud project



### View and manage tour files

* View recently edited and saved roaming files in the \[Recents] tab on the left-side navigation pane or in the \[All Projects] tab. Each folder contains tours from one project. You can view the tour files after entering the folder.
* Click the thumbnail or menu button of the tour file to view, edit, delete, share, etc.



**Project management**



Click the project thumbnail to open the project folder, which contains the uploaded Panorama Tour, Space Tour, and XR Tour. Click the menu icon in the lower-right corner to rename, modify the thumbnail, or delete items.

Click the project thumbnail to view the file details and share the project in the upper right corner. It's possible to download PLY files of XR tours. Click the menu in the lower right corner to edit and delete items.

### Share tours

Click the Share button in the upper-right corner of the tour file. In the pop-up window, copy the link and share the tour with others for viewing.









### Delete a Cloud Project  

1. Who can delete a cloud project

* Team Space: administrator, owner, project creator
* Group Space: administrator, owner, project creator

2. After deleting the cloud project, relevant data, including images, comments, virtual tours, etc., will be deleted as well.
3. After successful deletion, the Cloud storage space occupied by the project will be released.
