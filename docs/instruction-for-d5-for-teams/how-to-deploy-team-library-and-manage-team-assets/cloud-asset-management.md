# Cloud Asset Management

D5's cloud services (eg. D5 Studio) all occupy your storage space. Each team is provided with a total of 100GB of shared cloud storage, and detailed usage information can be accessed through the team management backend.

You can check the detailed usage status in “[Team Management Back](https://myspace.d5render.com/login)[end](https://myspace.d5render.com/login) - Cloud Storage”.

> **Info:** Note:

**Members with a team role of 'Member'** do not have access to the 'Team Management Backend'.


* Click “＞”, to show details (Including file size for each cloud project in Team Space).
* Currently only supports to view file size details under certain categories (D5 Cloud & D5 Studio).

Team administrators can monitor each member's personal space usage within the team and remove historical data of former team members to optimize storage capacity.

> **Info:** Note:

**Storage calculation includes My Space for all members & Team Space.** You can release your storage space by deleting historical data that is no longer in use, or contact us via sales for more cloud storage space.
